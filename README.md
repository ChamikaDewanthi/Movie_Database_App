# Movie Database App

The Movie Database App is an Android application that displays a list of movies with essential details such as title, release year, genre, and poster image. The app is designed with a focus on simplicity, usability, and efficient data handling, making it easy for users to browse and explore movie information.

The application reads movie data from a local JSON file, parses it, and dynamically presents it in a scrollable list using modern Android UI components. It also includes error handling and validation to ensure smooth performance even when data is incomplete or inconsistent.

## Project Execution

1. Clone this repository
2. Open the project in Android Studio
3. Let Gradle sync the project
4. Run the application on an emulator

## Installation as an App

1. Download the APK file from this repository.
2. Transfer the APK to your Android device.
3. Enable "Install from unknown sources" in device settings.
4. Open the APK file and install the application.
